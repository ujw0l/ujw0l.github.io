Library initialization

new ctcCarousel('selector1,selector2', { autoPlay: true, autoPlayInterval: number, autoPlaySelector: 'selector' });

First Parameter : one or multple selector to apply carousel.

Second Parameter :

autoPlay : boolean, (Optional, set true to activate slide autoplay)

autoPlayInterval : number, (Optional,number for slide interval in autoplay default 1sec e.g use 1000 for 1 second , 2000 for 2 seconds and so on )

autoPlaySelector : 'selectors', (one or all of the selector to set on autoplay (if left empty all carousel will autoplay)

Note 1 : Initialize carousel like above inside script tag on document ready or footer (Selector should be warapper element which contains images for carousel)

Note 2 : Do not forget to set height and width of wrapper element.
