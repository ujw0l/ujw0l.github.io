Library initialization

new new imageCarousel(selector/s,{callBack:callback function})

First Parameter : one or multple selector to apply carousel.

Second Parameter :(Js object)

callBack : function, (Optional, function to call after carousel is set, get selected element as a parameter)

Note 1 : Initialize carousel like above inside script tag on document ready or footer (Selector should be warapper element which contains images for carousel)

Note 2 : Do not forget to set height and width of wrapper element for better result.
